# OpenDesign Icon Logo Prompt

Generate a single **icon-only logo** (no text) for **OpenDesign**.

## What is OpenDesign?

An AI-powered UI mockup generator that transforms text descriptions into beautiful, production-ready app designs. Users describe their app idea in plain English, and AI generates polished mobile/desktop UI mockups with real-time streaming.

**Target users**: Indie developers, startups, non-technical founders, designers who want rapid UI prototyping.

**Core values**: Privacy-first, open source, user control, speed, simplicity.

## Brand Style

- **Aesthetic**: Editorial/magazine style (think Stripe Press, Linear, Notion)
- **Vibe**: Sophisticated, minimal, warm, premium, trustworthy
- **Primary color**: Warm Terracotta `#B8956F`
- **Background**: Transparent
- **Feel**: Modern but timeless, refined not flashy

## Icon Requirements

**Create ONE icon image:**
- **NO TEXT** - just a symbol/icon
- Clean, geometric, minimal design
- Uses terracotta (#B8956F) as the main color
- Square format
- Works as a favicon, app icon, and website logo

**The icon should subtly convey:**
- Design/UI creation (screen, frame, artboard, canvas)
- Openness (open source philosophy)
- Simplicity and speed

**Style rules:**
- NO gradients, glows, shadows, or 3D effects
- NO clich√© AI imagery (robots, neural networks, brains)
- NO cold blue tech colors
- NO text or letters
- Keep it flat, clean, and refined
- Must look good at 16px (favicon) and 512px (large display)

## Output

One single square icon, no text, on a transparent background, in PNG format.
